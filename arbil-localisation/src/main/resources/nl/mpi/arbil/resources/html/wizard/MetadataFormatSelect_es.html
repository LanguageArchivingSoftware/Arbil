Seleccionar un formato para los metadatos

Seleccionad el formato de metadatos que queréis utilizar. Ambos formatos se pueden utilizar
simultáneamente en Arbil, pero la mayoría de los usuarios utiliza normalmente uno del formatos aquí abajo. Al seleccionar
sólo una opción no se deshabilitará de manera permanente la otra.

IMDI significa Isle Meta Data Initiative standard. Para más informaciones
consultar el sitio www.mpi.nl/IMDI/.

CMDI is the Component MetaData Infrastructure, a profile based approach to
metadata where domain specific profiles can be designed that define a structure for metadata
instance. For more information, visit www.clarin.eu/cmdi.
