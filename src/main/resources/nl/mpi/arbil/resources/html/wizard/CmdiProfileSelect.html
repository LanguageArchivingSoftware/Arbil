Select your CMDI profiles

Please select the CMDI profiles that you wish to use.
You can always change your selection by selecting 'Templates/Profiles' from the Options menu.

Select the profiles you would like to start using from the list below. If this is the first time
you run this Wizard, the profiles will have to be download first. In this case, it will take some
time before the list appears.

If you hover over any one of the profile names, you will see a description of that profile.
