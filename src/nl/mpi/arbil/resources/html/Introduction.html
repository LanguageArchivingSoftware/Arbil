Linorg is a local metadata organising tool for researchers.
Corpus can be cached locally and new data added while off line.
This application is still in development so some areas are incomplete.

There are five main areas in the application:

  1. Top left tree listing contains remote corpus
  2. Middle left tree listing will contain off line copies of corpus
  3. Bottom left tree listing lists local directories
  4. Top right shows a preview of any selected IMDI
  5. Bottom right is where IMDI files can be viewed and edited


As a brief introduction to the application please follow the tasks below.
  1. Browse the remote corpus to find an area where your files will be uploaded to.
    These locations can be added or deleted via the mouse menu.
    If the corpus server that you work with is not listed then it can be added via the mouse menu.
    If there is a corpus that you regularly work on you can select it and copy the url then add it as a new location for easier access.
  2. cache a remote corpus locally
    if the branch has not been loaded then it may take some tome to load
    also note that , preferably one without too many nodes within it copy a remote branch locally

  3. select a local directory in the settings
    browse the directory
    check if the local files are in the branch copy via the icons

  4. select some nodes and view in a window
    select then remove some rows
    customise the column view

  5. select some corpus in the tree then open child nodes
    select some corpus and open in a window then select some child nodes to display in the grid

  6. select some nodes and open in a window then select some rows and copy them and paste into a spreadsheet application
