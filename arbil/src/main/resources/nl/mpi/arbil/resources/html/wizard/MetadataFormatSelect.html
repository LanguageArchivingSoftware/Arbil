Select a metadata format

Please select the metadata format you plan to use. Both metadata formats
can be used simultaneously in Arbil, but most users will only use one of
the formats below. Selecting only one option will not permanently
disable the other in any way.

IMDI is the Isle Meta Data Initiative standard. More
information can be found at www.mpi.nl/IMDI/.

CMDI is the Component MetaData Infrastructure, a profile
based approach to metadata where domain specific profiles can be designed
that define a structure for metadata instance. For more information,
visit www.clarin.eu/cmdi.
