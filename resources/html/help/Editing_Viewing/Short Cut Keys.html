General short cut keys
ctrl w Close top most window
ctrl tab next window
ctrl shift tab previous window
enter (in the tree) open the current tree selection in a table
ctrl p hide / show the selection preview
ctrl enter (when initiating a table cell edit via the keyboard) to edit a table cell in a long field editor
ctrl mouse click (when initiating a table cell edit via via the mouse) to edit a table cell in a long field editor
delete key delete the selected node in the tree
