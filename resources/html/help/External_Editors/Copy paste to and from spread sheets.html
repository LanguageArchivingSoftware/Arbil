Copy paste to and from spread sheets
Copy and paste can be used to move data from the application to external editors and then back into the application.
When copying from a table the format of the data will depend on what is selected. If multiple IMDI rows are selected and copied the data format cannot be pasted back into the application, however it can still be viewed and edited in the external editor.
If fields from a single IMDI are copied then it can then be pasted back into the application, providing the data maintains the same formatting.
